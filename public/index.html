JEPSO-CI Hook

You can set up GitHub with this url as a post-receive hook and JEPSO-CI will automatically re-test your library every time you push changes. To do this, go to your GitHub page and click on settings, you then need to add https://hook.jepso-ci.com. It should then look like:
